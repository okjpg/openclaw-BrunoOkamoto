3
OpenRouter — Camada de Experimentação
🔀
O que é o OpenRouter?

Uma API única que dá acesso a 300+ modelos de diferentes provedores. No curso, ele entra como camada de experimentação: você mantém OpenAI como base e usa OpenRouter quando quiser comparar Gemini, Claude, Llama e outros modelos sem trocar o fluxo principal. Importante: existe markup sobre parte dos preços, então use pela conveniência e pela flexibilidade, não como default obrigatório.

Por que usar OpenRouter?

✅ Uma API key pra tudo
✅ Comparar modelos sem mexer no setup principal
✅ Acesso a modelos open-source gratuitos
✅ Dashboard de uso unificado
⚠️ ~5% de markup sobre o preço do provider

Como configurar no OpenClaw:

# Configurar OpenRouter para testes opcionais
openclaw config set \
  openrouter.apiKey sk-or-v1-...

# Usar modelo via OpenRouter
openclaw config set model \
  openrouter/google/gemini-flash-1.5

Ferramentas Especializadas

LLMs fazem texto. Mas pesquisa web, scraping e geração de imagem precisam de ferramentas específicas — cada uma com seu modelo e custo ideal.

Pesquisa web em tempo real com fontes citadas. Diferente de um LLM comum, o Perplexity vai buscar informação atualizada na internet e entregar com links verificados. Ideal para monitorar notícias, checar preços, pesquisar concorrentes. Nossa skill deep-research-protocol usa Perplexity como backbone.

Como usamos no OpenClaw

Skill deep-research-protocol: pesquisa profunda estruturada sobre qualquer tema. Útil como experimento complementar, não como base do stack do curso. Uma query Sonar Pro custa ~$0.02 e entrega resultado com 10+ fontes citadas. Para um agente que faz 5 pesquisas por dia = ~$3/mês. Muito mais barato e confiável que pedir pro LLM "inventar" informações.

Apify — Scraping de Redes Sociais
Instagram, LinkedIn, Twitter/X bloqueiam IPs de cloud — não tem jeito de raspar dados diretamente. O Apify funciona como proxy: você chama um "Actor" (script de scraping), ele roda em IP residencial e devolve os dados. Usamos para métricas de conteúdo, benchmarking e análise de concorrentes.

O Perigo Oculto — Heartbeats em Volume
⚠️
Heartbeats parecem baratos — até você multiplicar

Um heartbeat custa centavos. Mas um agente com heartbeat a cada 30 minutos faz 48 chamadas por dia, 1.440 por mês. Com o modelo errado, isso vira o maior item do seu custo.

Simulação: 48 heartbeats/dia × 30 dias = 1.440 chamadas/mês

Configuração & Proteção

Setup recomendado em 5 comandos:

# 1. Modelo padrão para conversas
openclaw config set model openai/gpt-5.4

# 2. Modelo econômico para heartbeats (automático)
openclaw config set heartbeat.model openai/gpt-4o-mini

# 3. OpenRouter como opção alternativa
openclaw config set openrouter.apiKey sk-or-v1-...

openclaw config set rateLimit.requestsPerHour 100

# 5. Verificar gasto acumulado
openclaw usage report

Faça isso agora no platform.openai.com

Settings → Billing → Usage Limits → Monthly spend limit: $50 + Notification threshold: $25. Dois minutos que podem economizar centenas de dólares se um loop maluco acontecer.

Dúvidas Frequentes
Preciso de conta separada para o OpenRouter?
Sim — crie em openrouter.ai. Você deposita crédito lá e usa qualquer modelo com uma única API key. É opcional, mas muito prático se você quiser diversificar modelos sem múltiplas contas.
Haiku é menos capaz que Sonnet — vai travar no heartbeat?
Não para tarefas de heartbeat. Checar emails, verificar calendário, classificar notificações — Haiku faz isso perfeitamente. Onde Haiku perde é em raciocínio multi-step complexo e criatividade. Para 90% das tarefas de automação, é mais que suficiente.
Quando vale usar Perplexity vs só pedir ao LLM pra pesquisar?
Use Perplexity quando precisa de informação atualizada (notícias, preços, lançamentos) ou com fontes verificadas. LLMs têm data de corte e "inventam" fatos — Perplexity vai na web e cita as fontes. Para pesquisa séria, vale os $0.02 por query.
GPT-5.4 vale o preço?
Depende. Input a $2.50/M é razoável para um flagship — o problema é o output a $15/M. Para conversas longas ou sub-agentes verbosos, o custo sobe rápido. Use Claude Sonnet ou GPT-4o para 95% das tarefas; reserve o GPT-5.4 para contextos muito longos ou quando a qualidade extra justifica o custo.
Como saber exatamente quanto gastei hoje?
openclaw usage report mostra breakdown por modelo. O console de cada provider (anthropic.com, platform.openai.com) mostra gráficos em tempo real por dia e por modelo.
